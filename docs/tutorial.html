Tutorial / チュートリアル

ここでは、SivBoardの使い方について一緒に学んでいくよ！

まずは、Siv3Dのチュートリアルのおさらいからだな。

#1 : SivBoardの基本 / System, Print, Profiler #2 : 図形を描く / Circle, Rect, Line, Shape2D, etc. #3 : 動きを作る / Stopwatch, Periodic, Easing #4 : あたり判定 / CursorStyle #5 : 画像を描く / Texture #6 : データ構造 / Array, Grid, String #7 : 文字列と数値の置換 / Format, Parse #8 : フォントを使う / Font #9 : GUI / SimpleGUI #10 : シーンとウィンドウ / Scene, Window #11 : キーボード入力 / Key #12 : マウス入力 / Cursor #13 : オーディオ再生 / Audio #14 : ランダム / Random #15 : 2Dレンダ―ステート / ScopedRenderStates2D #16 : エフェクト #17 : アセット管理 / FontAsset, AudioAsset, TextureAsset #18 : テキストファイル / TextReader, TextWriter #19 : 設定ファイル / CSVData

SivBoardでは、Siv3Dの一部の機能しか扱えないけど、内容は盛りだくさん！
最新の機能に興味が出てきたら、Siv3Dのプログラミングにも挑戦してみてね。

よーし、だいぶレベルアップしたぞ！
次は、Siv3Dのチュートリアル以外の機能の確認だな。

#B1 : 時間と日付 / Time, Date, DateTime #B2 : ファイルシステム / FileSystem #B3 : ダイアログ / Dialog #B4 : kd木 / KDTree #B5 : ネットワーク / TCPServer, TCPClient #B6 : 2Dパーティクルシステム / ParticleSystem2D #B7 : 集中線描画 / SaturatedLinework #B8 : ジオメトリ / Geometry2D #B9 : マイク / Microphone, FFTResult #B10 : ZIPアーカイブの読み込み / ZIPReader

うーん、おなかいっぱいだ～！

ここまでマスターしたら、いろいろできる様になっているんじゃないかな。
最後に、Siv3Dと互換性のないその他の機能について紹介するよ！

ここからは、ボクが手伝うよ！

#C1 : 縦書きテキスト / VerticalText #C2 : 吹き出し / NgonBalloon, NStarBalloon #C3 : タイムイベント / TimeEvent #C4 : エフェクト / RingEffect, NumberEffect #C5 : 画像のアニメ―ション / MultiTextureAsset #C6 : シーンの管理 / SceneManager #C7 : スコアの保持 / ScoreAsset

これで、チュートリアルはおしまい。
紹介しきれていない機能については、ヘルプをチェックしてみてね！
