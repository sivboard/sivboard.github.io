Siv Board

Siv Boardは、手軽にSiv3Dでアートを楽しむソフトウェアです。

Siv3Dのコードを書いたスクリプトファイルを読み込むことで、
図形を描いたり、画像を表示して動かしたりできます。

インストールは不要で、任意の場所に置くだけで使えます。
以下、ふた通りの使い方が出来るように設計しています。

１．F5を押してスクリプトファイルを読み込む。
２．同じディレクトリにscript.asを置いて起動する。

まずは、下の方にある Script File Format のコードをコピーして、
script.asの名前で保存し、読み込んでみてください。
